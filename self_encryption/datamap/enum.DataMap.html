pub enum DataMap {
    Chunks(Vec<ChunkDetails>),
    Content(Vec<u8>),
    None,
}

Holds the infomation that required to recover the content of the encrypted file Depends on the file size, such info can be held as vector of ChunkDetail, or as raw data directly

Variants

Chunks

if file was large enough (larger than 3072 bytes, 3 * MIN_CHUNK_SIZE) this holds the list of chunks' info to decrypt it

Content

very small files (less than 3072 bytes, 3 * MIN_CHUNK_SIZE) are put here in entirely

None

empty datamap
